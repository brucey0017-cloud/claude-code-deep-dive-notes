# 13 - BUDDY System

> Claude Code 内置电子宠物系统

## 概述

BUDDY 是 Claude Code 的彩蛋功能，一个内置的电子宠物系统。

## 物种

共有 **18 种不同物种**：

### 常见物种
- 🐱 Cat - 猫
- 🐕 Dog - 狗
- 🐹 Hamster - 仓鼠
- 🐰 Rabbit - 兔子
- 🐍 Snake - 蛇

### 稀有物种
- 🐉 Dragon - 龙
- 🦄 Unicorn - 独角兽
- 🦊 Fox - 狐狸
- 🐼 Panda - 熊猫

## ASCII 艺术

每种宠物都有独特的 ASCII 艺术表示：

```
  /\_/\  
 ( o.o ) 
  > ^ <
```

## 互动

- **喂食**: `/buddy feed`
- **玩耍**: `/buddy play`
- **状态**: `/buddy status`

## 成长系统

- 活跃使用会提高宠物心情
- 长期不活跃会导致宠物不开心
- 完成任务可能获得奖励

## 设计意图

- 增加使用乐趣
- 情感连接
- 工作间歇的轻松时刻
